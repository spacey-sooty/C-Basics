## Dependencies
- Zig

## Setup
Run these commands to clone, compile and build this repository.
```sh
git clone https://github.com/spacey-sooty/C-Basics.git
cd C-Basics
chmod +x build
./build
./a.out
```
or on Windows
```ps1
git clone https://github.com/spacey-sooty/C-Basics.git
cd C-Basics
./build
./a.out
```
